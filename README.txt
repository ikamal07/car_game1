Thomas Racer - PWA
Files:
- index.html
- style.css
- main.js
- manifest.json
- service-worker.js
- assets/ (player.png, bg-music.wav, bump.wav, power.wav)
- icons/ (icon-192.png, icon-512.png)

How to run:
1. Unzip and serve the folder with any static server. For local testing you can use:
   python3 -m http.server 8000
   then open http://localhost:8000
2. Add to home screen on mobile to install as PWA.

Notes:
- The game is a simple top-down lane runner with obstacles and power-ups.
- Background music and short sound effects are included as generated WAVs.
